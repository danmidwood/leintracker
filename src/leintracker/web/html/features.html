Why you should use LeinTracker

  • Leiningen Projects

    LeinTracker can read your Leiningen project files, analyse them, and notify you about the freshness of your application dependencies.

  • Github Integration

    With LeinTracker's GitHub integration your Leiningen projects can be easily added. There are even build badges available for your READMEs.

  • Email Reports.

    LeinTracker email reports can notify you about updates to libraries you depend on. The reports are configurable in scope and delivery rate.

  • RESTful API.

    LeinTracker exposes services allowing you to push new projects for analysis, manage them and then pull dependency reports.

  • Looks Great.

    LeinTracker web is built using responsive design to look great on desktops, build screens and mobile devices.
